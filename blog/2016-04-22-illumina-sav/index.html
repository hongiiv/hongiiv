Sequencing Analysis Viewer (SAV)

The Sequencing Analysis Viewer (SAV) is an application that allows you, in real time, to view important quality metrics generated by the real-time analysis (RTA) software on the Illumina sequencing systems.

SAV에서 사용하는 파일들은 InterOp 디렉토리에 바이너리 형태로 SAV Viewer를 통해서 볼 수가 있다. 물론 윈도우즈용으로 제작되어 배포된다. 대부분 분석 결과에 대한 리포트를 제공하는 경우 fastq 파일에서 부터 시작하는 경우가 많은데, sequencing 수행중 장비 자체에서 만들어내는 정보를 포함한다면 좀 더 멋진 리포트가 될 수 있지 않을까 한다.

  • Quallity Metrics: This file contains the quality score distribution.
  • Error Metrics: This file contains cycle error rate and counts for perfect reads and read with 1–4 errors.
  • Tile Metrics: This file contains aggregate or read metrics by tile.
  • Corrected Intensity Metrics: This file contains base call metrics.
  • Image Metrics: This file contains minimum and maximum contrast values for image.
  • Index Metrics: This file reports the indexes count and is not generated in RTA 2.x
